## Installing

Install the component

```cli
npm install @supabase/ui
```

## Getting started

Import the component

```js
import { Auth } from '@supabase/ui'
```

A community maintained Svelte version of this Auth component is also available:
[joshnuss/supabase-ui-svelte](https://github.com/joshnuss/supabase-ui-svelte)
